# الإصدار الأول من `indexnow-notifier` على NPM

اليوم أطلق **indexnow-notifier**، وهي أداة CLI صغيرة تحل مشكلة محددة جدًا كانت تزعجني أثناء النشر.

عندما تتحدث الصفحات، يجب أن تعرف محركات البحث بذلك فورًا.  
ليس غدًا.  
وليس بعد انتظار الزاحف لزيارة موقعك "في النهاية".

وهنا يأتي دور IndexNow.

وبصراحة، إرسال إشعارات IndexNow يدويًا في كل عملية نشر يبدو من نوع المهام التي يعد المطورون أنفسهم بأتمتتها "لاحقًا". ونعرف جميعًا كيف تنتهي هذه القصة.

لذلك بنيت **indexnow-notifier**.

## ما هو `indexnow-notifier`؟

`indexnow-notifier` أداة CLI خفيفة مصممة لخطوط CI/CD ترسل تلقائيًا عناوين URL المحدثة إلى خدمات IndexNow بعد النشر.

يدعم حاليًا:

- Next.js Pages Router
- Next.js App Router

ويجري التخطيط لدعم:

- React Router
- Laravel
- Angular

الهدف بسيط.

انشر تطبيقك.  
أرسل إشعارًا إلى IndexNow تلقائيًا.  
اذهب واشرب قهوتك بينما تقوم الروبوتات بما يلزم.

## لماذا بنيته

كنت أريد شيئًا مباشرًا لخطوط النشر.

لا إعداد معقد.  
لا إطار ضخم.  
لا "حل مؤسسي" يحتاج بطريقة ما إلى 14 ملف إعداد ومستشار.

فقط أمر بسيط يمكن تشغيله داخل GitHub Actions أو GitLab CI أو Jenkins أو أي آلة تحتجز عملية النشر لديك.

هذه الأداة موجودة لأن أتمتة SEO ينبغي أن تبدو مملة بأفضل معنى ممكن.

## التثبيت

يمكنك تثبيته عالميًا باستخدام npm:

```bash
npm i -g indexnow-notifier
```

أو تشغيله مباشرة عبر npx:

```bash
npx indexnow-notifier
```

يمكنك أيضًا استنساخ المستودع وتثبيته يدويًا:

```bash
git clone https://github.com/mrepol742/indexnow-notifier
cd indexnow-notifier
npm i -g .
```

لمستخدمي Linux، قد تحتاج إلى `sudo` عند التثبيت العالمي:

```bash
sudo npm i -g indexnow-notifier
```

لأن Linux يحب أن يذكرنا من هو صاحب الكلمة.

## مصمم لخطوط CI/CD

حالة الاستخدام الأساسية هي أتمتة النشر. بعد نشر تطبيقك، يمكن لـ `indexnow-notifier` إبلاغ خدمات IndexNow بالصفحات المحدثة فورًا.

هذا يساعد محركات البحث على اكتشاف المحتوى الجديد بسرعة أكبر من دون انتظار عودة الزواحف إلى موقعك بشكل طبيعي.

مفيد خصوصًا لـ:

- المواقع التي تتحدث كثيرًا
- المدونات
- مواقع الوثائق
- صفحات التجارة الإلكترونية
- المحتوى الإخباري
- المشاريع التي تركز على SEO

باختصار، إذا كانت صفحاتك تتغير كثيرًا، فهذه الأداة تساعد محركات البحث على مواكبة ذلك.
